# Task Manager App

A full-stack MERN Task Manager application.

## Features
- User Registration
- User Login Authentication
- JWT Authentication
- Create Tasks
- Update Task Status
- Delete Tasks
- Task Filtering

## Tech Stack
- React.js
- Node.js
- Express.js
- MongoDB Atlas
- JWT Authentication

## Installation

### Backend
```bash
npm install
npm run dev
```

### Frontend
```bash
npm install
npm run dev
```
